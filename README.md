# customer_behaviour_analysis
Data analytics project showcasing behaviour analysis using python ,SQL and power BI.
📊 Data Analytics Project
📌 Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data processing to generating business insights. It involves data loading, cleaning, exploratory data analysis (EDA), SQL-based querying, and building an interactive dashboard for visualization.

The goal of this project is to extract meaningful insights and present them in a structured and visually appealing format for decision-making.

📂 Dataset
The dataset contains customer-related transactional and behavioral data.
It includes attributes such as demographics, purchase details, and transaction history.
The dataset is loaded and processed using Python for further analysis.
🛠️ Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
PostgreSQL (for structured querying)
Power BI (for dashboard and visualization)
Jupyter Notebook (for development and analysis)
🔄 Project Workflow / Steps
Data Loading
Imported dataset using Pandas
Checked structure, data types, and basic information
Exploratory Data Analysis (EDA)
Analyzed distributions, trends, and relationships
Identified missing values and anomalies
Used visualizations for better understanding
Data Cleaning
Handled missing values
Removed duplicates
Standardized column names and formats
Created new derived features where required
SQL Analysis (PostgreSQL)
Loaded cleaned data into PostgreSQL
Performed queries for:
Aggregations
Filtering conditions
Subqueries and joins
Extracted insights for reporting
Dashboard Creation (Power BI)
Connected Power BI with processed dataset
Designed interactive dashboards
Included KPIs, charts, and filters
📊 Dashboard

The Power BI dashboard provides:

Key performance indicators (KPIs)
Customer behavior insights
Sales trends and patterns
Interactive filters for dynamic analysis
